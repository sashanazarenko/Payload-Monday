# Sales Product Search - Admin-Only MVP WBS

## Scope definition

This WBS is strictly for:

- **Admin view only**
- **Admin role/user only**

Excluded from this file:

- Sales workflows
- Finance-only workflows (unless touched by Admin controls)
- Non-admin dashboards and routes

## Notes for implementation

- This file intentionally decomposes dashboard and product creation into granular rows per your request.
- Product creation is represented by **separate flow rows**: Standard, Bespoke, APPA-linked, and Proposal-Only.
- Personal profile management is separated from user administration:
  - **User Management** = Admin managing other users
  - **Personal Profile** = Admin managing own account/profile
